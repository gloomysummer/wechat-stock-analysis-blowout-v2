# 财经号合规写法清单

这份清单用于公众号财经文章发布前复核，默认纳入 `review.md`。

## 1. 标题
- [ ] 未把预期写成事实
- [ ] 未出现“实锤合作 / 核心受益 / 还有空间 / 值得布局 / 建议买入”
- [ ] 标题更像“市场观察”，而不是“单票推荐”

## 2. 摘要与导语
- [ ] 明确区分公开事实与市场预期
- [ ] 若公司未公告合作/订单，已明确写出“截至当日未见正式公告确认”
- [ ] 没有使用容易诱导交易的措辞

## 3. 正文事实边界
- [ ] 主营业务、营收、净利润、现金流、负债率等均有来源
- [ ] 客户、订单、合作、份额、供应链地位等无公告不下结论
- [ ] 没有把媒体消息、传闻、市场联想改写成确定事实

## 4. 观点边界
- [ ] 没有买卖建议、收益承诺、目标价、涨跌预测
- [ ] 允许使用“市场在看什么 / 更稳妥的理解是 / 后续值得跟踪的是”
- [ ] 不用“后面还有空间 / 这就是机会 / 可重点关注”

## 5. 风险与平衡
- [ ] 写了支撑因素，也写了不确定因素
- [ ] 强判断后面有数据或公开资料支撑
- [ ] 能写“仍需观察”，就不写“已经成立”

## 6. 发布门槛
- [ ] `review.md` 明确写出“可发布”
- [ ] `review.md` 包含“关键事实核对清单”
- [ ] `review.md` 包含“财经号合规写法清单”
- [ ] 任一项不通过，必须先重写 `article.md` 再复审
