# 港股重大风险检索关键词矩阵

这份矩阵解决的是：当文章命中港股退市、私有化、纾困、债务展期等场景时，不能只搜几个显眼词，而要沿着“事件词 + 文件词 + 参与方 + 债务词 + 时间轴”一起扩展。

## 一、核心原则

不要只搜：
- `{公司名} 退市`
- `{公司名} 私有化`
- `{公司名} 纾困`

这三个词只能算起点，不够支撑完整的风险溯源。

正确做法是把检索词拆成 6 组：
1. 上市状态词
2. 交易结构词
3. 债务与信用词
4. 重组与纾困参与方词
5. 文件类型词
6. 时间轴词

## 二、基础检索词组

### A. 上市状态 / 结果词
- `{公司名} 退市`
- `{公司名} 撤销上市地位`
- `{公司名} 长期停牌`
- `{公司名} 复牌指引`
- `{公司名} delisting`
- `{公司名} withdrawal of listing`
- `{公司名} suspension`
- `{公司名} resumption guidance`

### B. 私有化 / 交易结构词
- `{公司名} 私有化`
- `{公司名} 协议安排`
- `{公司名} 要约`
- `{公司名} 联合公告`
- `{公司名} 通函`
- `{公司名} privatization`
- `{公司名} scheme of arrangement`
- `{公司名} offer document`
- `{公司名} composite document`
- `{公司名} whitewash waiver`

### C. 债务 / 信用 / 流动性词
- `{公司名} 债务违约`
- `{公司名} 债务展期`
- `{公司名} 流动性`
- `{公司名} 借新还旧`
- `{公司名} 商票逾期`
- `{公司名} 评级下调`
- `{公司名} debt restructuring`
- `{公司名} default`
- `{公司名} extension`
- `{公司名} liquidity pressure`
- `{公司名} rating downgrade`
- `{公司名} overdue notes`

### D. 纾困 / 重组参与方词
- `{公司名} 中国信达`
- `{公司名} 华融`
- `{公司名} 东方资产`
- `{公司名} 长城资产`
- `{公司名} AMC`
- `{公司名} 纾困方案`
- `{公司名} 财务纾困`
- `{公司名} restructuring support`

### E. 文件类型词
- `{公司名} 公告`
- `{公司名} 通函`
- `{公司名} 联合公告`
- `{公司名} 年报`
- `{公司名} 债权人安排计划`
- `{公司名} 清盘呈请`
- `{公司名} announcement`
- `{公司名} circular`
- `{公司名} annual report`
- `{公司名} scheme document`
- `{公司名} creditor arrangement`
- `{公司名} winding-up petition`

### F. 时间轴词（动态回看）
- 默认按**当前年份向前回看 5 年**，而不是写死某几个年份
- 可用：`{公司名} 最近5年`、`{公司名} 近五年`、`{公司名} 过去5年`
- 可用：`{公司名} {当前年份}`、`{公司名} {当前年份-1}`、`{公司名} {当前年份-2}` ……一直回看到 `当前年份-5`
- 可用：`{公司名} 当年退市`、`{公司名} 退市前后`、`{公司名} before delisting`、`{公司名} after delisting`
- 如果重大风险链条发生在 5 年之前，但今天仍然影响判断，就继续向前追，不受 5 年上限限制

## 三、推荐组合检索公式

### 时间轴默认规则
- 默认从当前年份往前回看 5 年，不写死成 2019、2020、2021 这种固定年份
- 如果公司重大风险链条明显早于过去 5 年，但仍然影响今天判断，应继续向前延长时间轴
- 先按“最近5年”粗搜，再按关键年份精搜

### 1. 先确认上市状态
- `{公司名} + 退市 / 撤销上市地位 / 停牌 / 复牌指引`
- `{公司名} + delisting / suspension / resumption`

### 2. 再确认私有化或交易结构
- `{公司名} + 私有化 + 联合公告`
- `{公司名} + 协议安排 + 通函`
- `{公司名} + privatization + scheme of arrangement`

### 3. 再确认债务和纾困背景
- `{公司名} + 债务展期 / 债务违约 / 评级下调`
- `{公司名} + 中国信达 / 华融 / AMC + 纾困`
- `{公司名} + debt restructuring / default / rating downgrade`

### 4. 再追深层机制
- `{公司名} + 扩张 + 债务压力`
- `{公司名} + 对外担保 / 互保 / 资金链`
- `{公司名} + expansion + leverage`
- `{公司名} + guarantee chain / mutual guarantee`

### 5. 最后回到今天
- `{公司名} + 现在 + 债务`
- `{公司名} + 最新 + 公告`
- `{公司名} + current + debt`
- `{公司名} + latest announcement`

## 四、针对你这个场景的额外建议关键词

如果想让模型更容易追到你说的那条链，我建议重点把下面这些词也纳入：

### 1. 退市类不止“退市”
还要一起搜：
- 撤销上市地位
- 长期停牌
- 复牌指引
- 停牌更新
- 协议安排
- 要约
- 私有化

### 2. 纾困类不止“纾困”
还要一起搜：
- 债务重组
- 展期
- 违约
- 信用事件
- 流动性压力
- 战略投资者
- 资产出售
- 债权人安排计划

### 3. 深层机制类建议加这些词
如果想追更深层原因，建议引入：
- 扩张
- 多元化投资
- 高杠杆
- 对外担保
- 互保
- 关联担保
- 资金链
- 地方信用风险
- guarantee chain
- mutual guarantee
- leverage

### 4. 参与方类也很关键
很多故事不是从公司名直接搜出来的，而是从参与方反推：
- 中国信达
- 华融
- 东方资产
- 长城资产
- 债权人委员会
- 财务顾问
- 清盘人 / 临时清盘人

## 五、不要只搜中文
港股很多关键文件和媒体表达是英文的，建议同一轮至少补 1-2 组英文词：
- delisting
- privatization
- scheme of arrangement
- suspension
- resumption guidance
- debt restructuring
- default
- rating downgrade
- winding-up petition

## 六、不要只搜公司名
最好同时带：
- 股票代码
- 历史简称
- 母公司 / 集团名
- 纾困方或收购方名称

例如：
- `{公司名} + 股票代码 + privatization`
- `{公司名} + 集团名 + 债务展期`
- `{公司名} + 中国信达 + 联合公告`
- `{公司名} + 最近5年 + 债务重组`
- `{公司名} + {当前年份-3} + delisting`

## 七、目标效果

这套矩阵的目标不是搜得更杂，而是让模型更容易抓到：
- 结果事实
- 事件载体
- 资金背景
- 重组参与方
- 深层放大机制
- 当前是否出清
