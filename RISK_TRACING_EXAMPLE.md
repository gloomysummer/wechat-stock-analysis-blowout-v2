# 重大风险溯源写法示例

这份示例不是让文章照抄，而是告诉模型：当识别到重大风险时，应该怎样把风险写成一条因果链，而不是一组标签。

## 示例场景：历史退市 / 私有化 / 债务纾困类风险

### 错误写法

- 公司曾在某个阶段退市，存在债务风险。
- 当时公司经营承压，后续被私有化。
- 这说明公司风险较大。

问题：
- 只有结果，没有原因
- 没有解释事件之间的关系
- 没有区分直接原因和更深层机制
- 读者看完仍然不知道“为什么会走到这一步”

### 更好的写法

先看结果。对一家消费品公司来说，普通的业绩波动并不稀奇，但如果它最终走到了私有化退市这一步，性质就完全不一样了。因为退市本身不是日常经营波动，而是一个足以说明资本结构和风险状态已经发生变化的结果。

如果继续往下追，退市往往不是第一张多米诺骨牌，而是后面的结果。更值得关心的是，它为什么会走到这一步。公开资料通常会先给出一个直接原因，比如纾困方案、债务展期、控制权调整、资产重组。这一层写的是“事件是怎么落地的”。

但只写到这里还不够。再往下一层，文章还要追问：为什么会出现纾困、展期或私有化？这时就要进入中层原因，比如债务结构过重、扩张节奏失衡、担保链条过长、现金流缓冲不足。这一层写的是“公司内部哪一层机制先出了问题”。

如果素材继续支持，还可以再追一层更深的机制。比如某些公司表面上是扩张失衡，背后却叠加了地方互保、行业信用收缩、关键客户回款恶化、资本运作失败等外部因素。写到这一层时必须特别注意证据边界：如果这是公告和交易所文件明确写到的，可以直接写；如果更多来自多家媒体梳理，可以写“公开报道显示”；如果只是市场长期存在的一种解释，就只能写“市场曾有一种看法”，不能写成公司已经确认的结论。

最后，风险段不能只停在历史复盘。真正成熟的写法，还要再落回今天：当年那条风险放大链条，现在有没有被切断？如果债务结构已经出清、担保链条已经解除、控制权已经稳定，那历史风险的参考价值就更多体现在风险偏好上；但如果类似的融资依赖、扩张冲动或外部连带机制仍然存在，那它就不只是历史故事，而是今天仍然有效的风险变量。

### 这类段落的标准结构

1. 先点异常结果：退市、私有化、违约、问询、处罚
2. 再写直接原因：纾困方案、债务展期、评级下调、控制权安排
3. 再写中层原因：扩张、债务结构、担保、现金流、治理问题
4. 再写更深层机制：地方互保、行业环境、客户结构、资本运作、监管变化
5. 最后回到今天：这条机制现在还在不在

### 证据边界示例

- A 级证据写法：公司公告显示、交易所文件显示、评级报告指出
- B 级证据写法：公开报道显示、多家媒体提到、市场普遍关注的是
- C 级证据写法：市场曾有一种解释、公开讨论中存在这种看法

### 目标效果

读者看完后，不只是知道“公司有风险”，而是能明白：
- 风险到底是怎么一步步形成的
- 哪个环节是最关键的放大器
- 这件事今天到底过去了，还是没过去
