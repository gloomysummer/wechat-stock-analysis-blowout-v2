# A股 / 港股通用重大风险检索矩阵

这份矩阵解决的是：不管公司在 A 股、港股，还是历史上跨市场上市，只要文章需要写“真正有分量的风险”，就不能只停留在“风险、负面、问题”这种泛词，而要系统扩展到控制权、监管、信用、法律、审计、经营异常等高价值风险信号。

## 一、核心原则

不要只搜：
- `{公司名} 风险`
- `{公司名} 负面`
- `{公司名} 问题`

这种搜法信息太散，容易抓到低价值舆情，反而漏掉真正重要的结构性风险。

正确做法是按 8 个风险维度去扩关键词：
1. 股东与控制权
2. 监管与退市
3. 法律与执行
4. 债务与信用
5. 票据与商票
6. 审计与财务边界
7. 经营异常
8. 资本运作与资产处置

## 三、推荐搜索公式

### 第一层：先抓结果事实
- `{公司名} + 质押 / 冻结 / 立案调查 / 诉讼 / 违约 / 审计意见 / 停产`

### 第二层：再抓事件载体
- `{公司名} + 公告 / 通函 / 问询函 / 处罚决定 / 裁判文书 / 评级报告`

### 第三层：再抓放大机制
- `{公司名} + 借新还旧 / 担保 / 互保 / 控制权 / 流动性 / 资产出售 / 纾困`

### 第四层：最后补市场解释
- `{公司名} + 主流财经媒体关键词`
- `{公司名} + 市场公开讨论中的关键词脉络`

## 四、优先级建议

### 高优先级，建议默认一起搜的词
如果篇幅有限，至少补这 12 个：
- 质押
- 冻结
- 控制权变更
- 立案调查
- 问询函
- 诉讼
- 被执行
- 债务违约
- 债务展期
- 商票逾期
- 审计意见
- 持续经营重大不确定性

### 命中这些词时，建议自动升级关注级别
- 退市 / 私有化 / ST / *ST
- 违约 / 展期 / 评级下调
- 立案调查 / 行政处罚
- 保留意见 / 无法表示意见 / 持续经营
- 高比例质押 + 冻结 + 控制权变化

## 五、跨市场写作提醒

- A 股和港股的风险词表面不同，但底层结构类似：控制权、信用、监管、诉讼、审计、经营异常
- 不要因为市场不同就只搜各自的表层词，要把底层机制搜全
- 同一家公司如果有 A 股、港股、集团层、子公司层信息，要交叉验证，不要只盯一个上市壳

## 六、目标效果

这套矩阵的目标是让文章更容易识别：
- 哪些是会影响估值和预期的结构性风险
- 哪些只是一般性噪音
- 哪些风险之间其实属于同一条因果链
