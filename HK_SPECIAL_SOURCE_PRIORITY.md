# 港股退市 / 私有化 / 纾困场景专项来源优先级规则

这份规则解决的是：当文章涉及港股上市历史、退市、私有化、纾困、债务展期等情形时，应该优先依赖哪些来源，按什么顺序查证，而不是泛泛抓网页后再拼故事。

## 一、什么时候启用这份规则

只要出现以下任一情况，就默认启用：
- 公司当前或历史上是港股上市公司
- 出现“退市、私有化、撤销上市地位、停牌、复牌指引、长期停牌、协议安排、要约、白洗豁免”等关键词
- 出现“纾困、债务重组、债务展期、违约、清盘呈请、债权人安排计划”等关键词
- 出现中国信达、华融、东方、长城等 AMC 参与纾困或重组的线索

## 二、来源优先级

### A 级：必须优先核实的官方来源

1. 港交所公告系统（HKEXnews）
- 公司公告
- 联合公告
- 私有化 / 要约 / 协议安排文件
- 复牌指引 / 停牌更新 / 退市相关公告
- 年报、中报、通函、债务重组更新

2. 港交所规则与指引
- 退市、长期停牌、复牌指引相关规则与指引
- 与撤销上市地位、协议安排、持续上市条件相关的规则背景

3. 香港证监会（SFC）
- 白洗豁免、特别交易同意、监管公告
- 与私有化、控制权变化、要约相关披露

4. 公司正式文件
- 年报、中报、通函、债务重组计划、债权人安排计划文件
- 与资产出售、私有化、融资安排相关的正式披露

### B 级：高可信辅助来源

5. 评级机构 / 债券 / 信用材料
- 评级报告、评级下调说明、债券存续期文件
- 公开披露的债务展期、违约、信用事件说明

6. 法院 / 执行 / 清盘相关公开资料
- 清盘呈请、法院裁决、执行信息、公开司法文件

7. 纾困方或重组参与方正式材料
- 如果中国信达、华融、东方、长城等 AMC 参与，优先看其正式公告或公开材料
- 如果银行、债权人委员会、财务顾问有正式披露，也应纳入

### C 级：可用但必须降级表达的来源

8. 主流财经媒体
- 财联社、第一财经、21财经、香港经济日报、明报财经、格隆汇、财华社、新浪港股等
- 只能作为对官方文件的补充解释，不应反客为主

9. 市场公开讨论与地方舆情
- 只能作为背景线索，用于提出“市场曾有一种解释”
- 若无 A/B 级来源支撑，不能写成确定性结论

## 三、推荐查证顺序

如果涉及港股退市 / 私有化 / 纾困场景，默认按以下顺序查：

1. 先确认公司当时和现在的上市状态
- 是否已退市
- 是主动私有化还是被动退市
- 是否经历长期停牌和复牌指引

2. 再确认异常结果的直接载体
- 私有化方案
- 协议安排 / 要约文件
- 债务重组安排
- 纾困方案或资产出售安排

3. 再确认为什么会走到这一步
- 当时是否已经出现债务违约、展期、评级下调、流动性压力
- 是否存在控股股东质押、担保、资产出售或控制权安排

4. 再看更深层的机制
- 扩张失衡
- 债务结构问题
- 担保链条 / 互保链条
- 行业信用收缩
- 关键客户或资产问题

5. 最后才用媒体和市场讨论补解释
- 媒体负责补“怎么理解”
- 市场讨论负责补“公开脉络中存在哪种解释”
- 但不能倒过来用媒体或讨论替代官方文件

## 四、写作边界

### 可以直接写成事实的内容
- 来自 HKEXnews、SFC、公司公告、评级材料、法院公开文件的内容

### 应写成“公开报道显示”的内容
- 来自主流财经媒体、且与官方文件相互印证的内容

### 只能写成“市场曾有一种解释”的内容
- 地方互保、隐性担保、非正式市场传闻、行业圈讨论
- 除非已被公告或主流媒体多源交叉验证，否则不能写成公司确认事实

## 五、特别提醒：长链条场景不要写短

像“退市 -> 私有化 -> 纾困 -> 债务展期或违约 -> 扩张失衡 / 互保链条”这种场景，绝不能只写一句“公司后来退市，存在债务问题”。

正确做法是：
- 先用 A 级来源确定退市 / 私有化 / 纾困是否真实存在
- 再用 A/B 级来源确定违约、展期、评级或债务压力是否存在
- 再谨慎判断更深层机制，必要时用“市场曾有一种解释”保留边界

## 六、目标效果

文章最终要做到：
- 港股重大风险的起点和终点都能说清楚
- 不是只会写“公司后来退市了”
- 能讲清退市、私有化、纾困、违约之间的先后关系
- 能把官方事实、媒体解释、市场脉络分层写清
