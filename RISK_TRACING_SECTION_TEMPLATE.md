# 重大风险溯源小节模板

这份模板解决的不是“要不要写风险”，而是“命中重大风险后，这一小节应该怎么写”。

## 什么时候使用

当文章识别到以下任一重大风险信号时，优先考虑使用这份模板来组织风险段中的一个完整小节：
- 退市、私有化、ST、*ST、退市风险警示
- 债务违约、展期、评级下调、商票逾期
- 高比例质押、冻结、控制权变化
- 重大诉讼、被执行、失信、资产冻结
- 立案调查、交易所问询、监管处罚
- 重大审计意见、持续经营疑虑、停产减产、重大减值

## 小节目标

读者看完这一小节后，应该清楚：
- 发生了什么异常结果
- 它为什么会发生
- 哪个环节是风险放大器
- 这件事今天到底过去了，还是没过去

## 先提醒一条：不要把内部提纲直接写给读者

模型内部可以按“异常结果、直接原因、中层原因、更深层机制、回到今天”思考。
但正文**默认不要直接写成**：
- 异常结果：……
- 直接原因：……
- 中层原因：……
- 更深层机制：……

这会让文章立刻变成报告腔，读起来很像 AI。

正确做法是：
- 只保留一个自然的小标题
- 用几段自然衔接的正文，把这几层意思写进去
- 让读者感觉是在被带着理解，而不是在看答题卡

## 推荐骨架

### 1. 小标题

小标题不要写成“风险提示”这种空标题，优先写成判断句或追问句。

示例方向：
- 它真正值得警惕的，不是利润波动，而是那次私有化退市
- 一次债务展期背后，到底是哪条链先断了
- 控股股东质押之外，真正的问题出在哪一层

### 2. 第一段：先点异常结果

先告诉读者，真正异常的结果是什么。

推荐写法：
- 对一家正常经营公司来说，普通业绩波动并不稀奇，但如果最终走到了 {退市 / 私有化 / 债务违约 / 立案调查} 这一步，性质就完全不同了。
- 因为这不是日常波动，而是足以说明资本结构、信用状态或经营安全边界已经发生变化的结果。

### 3. 第二段：再写直接原因

把事件为什么落地写清楚。

推荐写法：
- 如果继续往下追，这个结果通常不是第一张多米诺骨牌，而是后面的结果。公开资料先给出的直接原因，往往是 {纾困方案 / 债务展期 / 评级下调 / 控制权安排 / 监管行动}。
- 这一层写的是：事件是怎么落地的。

### 4. 第三段：再写中层原因

继续追问，为什么会走到需要纾困、展期或监管介入这一步。

推荐写法：
- 再往下一层，文章要追问的是：为什么会出现这一步？
- 这时就进入中层原因，比如 {扩张失衡 / 债务结构过重 / 担保链条过长 / 现金流缓冲不足 / 治理失衡}。
- 这一层写的是：公司内部哪一层机制先出了问题。

### 5. 第四段：再写更深层机制

如果素材支持，再写真正放大风险的背景机制。

推荐写法：
- 如果素材继续支持，还可以再追一层更深的机制。
比如 {地方互保 / 行业信用收缩 / 大客户回款恶化 / 资本运作失败 / 监管环境变化}，它们可能不是第一原因，但可能是风险被迅速放大的背景因素。

边界要求：
- A 级证据可直接写事实
- B 级证据写成“公开报道显示”“多家媒体提到”
- C 级证据只能写成“市场曾有一种解释”“公开讨论中存在这种看法”

### 6. 第五段：最后回到今天

不要停在历史复盘，要回到当前判断。

推荐写法：
- 真正重要的，不只是这件事历史上发生过，而是当年那条风险放大链条，现在有没有被切断。
- 如果相关机制已经解除，它更像是一段风险偏好的历史包袱；如果类似的融资依赖、扩张冲动或外部连带机制仍然存在，那它就不是过去式，而是今天仍然有效的风险变量。

## 简版模板

```md
## {判断句/追问句式小标题}

先看结果。对一家{行业}公司来说，普通的{业绩/经营}波动并不稀奇，但如果它最终走到了 {异常结果} 这一步，性质就完全不一样了。因为这不是日常波动，而是一个足以说明 {资本结构/信用状态/经营安全边界} 已经发生变化的结果。

如果继续往下追，{异常结果} 往往不是第一张多米诺骨牌，而是后面的结果。公开资料先给出的直接原因，通常是 {直接原因}。这一层写的是：事件是怎么落地的。

但只写到这里还不够。再往下一层，真正值得追问的是：为什么会出现 {直接原因}？这时就要进入中层原因，比如 {中层原因}。这一层写的是：公司内部哪一层机制先出了问题。

如果素材继续支持，还可以再追一层更深的机制。比如 {更深层机制}。写到这一层时必须特别注意证据边界：A 级可直接写事实，B 级写成公开报道，C 级只能写成一种公开解释。

最后，风险段不能只停在历史复盘。真正需要回到今天的问题是：当年那条风险放大链条，现在还有没有被切断？如果 {机制已解除}，那它更多是历史风险；如果 {机制仍存在}，那它就仍然是今天的风险变量。
```

## 禁止写法

- 不要只写“公司曾经退市/违约/被问询，所以风险很大”
- 不要把直接原因和深层原因混成一句话
- 不要把市场解释直接写成公司已确认结论
- 不要只写历史故事，不回到今天的影响
- 不要把一整段写成五条 bullet，优先写成自然段推进
